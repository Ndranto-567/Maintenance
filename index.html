🚀 Mise en service en 3 étapes

Cette application fonctionne avec 2 fichiers : index.html (GitHub Pages) + Code.gs (Google Apps Script).

  1. Google Sheets : Créez un Google Sheet avec les feuilles Base Liste Equipement, Base Unité d'intervention, Base Processus, Base Maintenance, Personnel Maintenance
  2. Apps Script : Dans le Sheet → Extensions → Apps Script → Collez le contenu de Code.gs → Déployer comme Web App (accès : Tout le monde) → Copiez l'URL
  3. Ici : Collez l'URL dans les champs ci-dessus → Sauvegarder
